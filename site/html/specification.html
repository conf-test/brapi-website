BrAPI V2 Developer Preview

CAUTION: This is a Developer Preview for BrAPI V2. BrAPI V2 has not been officially released. All V2 documentation below is subject to change without warning.
BrAPI V2 is broken into modules to help maintain organization and consistancy within the API.

V2.0
BrAPI Core

The BrAPI Core module contains high level entities used for organization and management. This includes Programs, Trials, Studies, Locations, People, and Lists

V2.0
BrAPI Phenotyping

The BrAPI Phenotyping module contains entities related to phenotypic observations. This includes Observation Units, Observations, Observation Variables, Traits, Scales, Methods, and Images

V2.0
BrAPI Genotyping

The BrAPI Genotyping module contains entities related to genotyping analysis. This includes Samples, Markers, Variant Sets, Variants, Call Sets, Calls, References, Reads, and Vendor Orders

V2.0
BrAPI Germplasm

The BrAPI Germplasm module contains entities related to germplasm management. This includes Germplasm, Germplasm Attributes, Seed Lots, Crosses, Pedigree, and Progeny
